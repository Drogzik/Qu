Папка модели

Если ставили через установщик с интернетом, модель TinyLlama (~0,7 ГБ) скачивается автоматически.
Её можно заменить своим файлом .gguf (удали старый или положи другой — программа возьмёт первый подходящий).

Положите сюда один файл .gguf рядом с Pomoshnik.exe, внутри папки «models».

Если ставили через установщик, путь обычно такой:
%LOCALAPPDATA%\Programs\Pomoshnik\models

Дополнительно программа ищет модели в:
  Документы\Pomoshnik\models
  Загрузки\Pomoshnik\models  (или Downloads\Pomoshnik\models)
Можно задать каталог в переменной окружения POMOSHNIK_MODEL_DIR.

После копирования файла достаточно отправить новое сообщение в чат (перезапуск не обязателен).

Пока файла модели нет: закладки, напоминания и команды в чате всё равно работают.
